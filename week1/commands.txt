1- To create a hidden file we use the command: attrib +h <file/folder name>
to display it again we use the command: attrib -h <file/folder name>
2- To create a multiple nested directories we use -p or -- parents with mkdir command, for example: mkdir -p year/month/week/day
3- To append a message to a file we use: cat >> file.txt , enter, then typing a message, and after finishing ctrl+z
